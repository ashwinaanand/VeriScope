# Fake vs Real Information Detector

## Overview
VeriScope is a simple web-based tool that helps users identify potentially misleading or fake information. Users can paste any message or text, and the system analyzes it to classify the content as real, fake, or uncertain.

The goal of this project is to make people more aware of misinformation and encourage responsible sharing of information.

## Problem
Misinformation spreads quickly through social media and messaging platforms. Many users are unable to verify whether the content they receive is reliable, which can lead to confusion and poor decision-making.

## Solution
This project analyzes input text using basic pattern and keyword detection to identify suspicious phrases often found in misleading messages. Based on this, it provides a simple classification of the content.

## Features
- Input any text or message  
- Detect suspicious patterns in content  
- Classify as real, fake, or uncertain  
- Simple and easy to use  

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## How It Works
The system checks for commonly used phrases such as:
- “forwarded as received”  
- “urgent share”  
- “this is not shown on news”  

Based on these patterns, it estimates the reliability of the message.
